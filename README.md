# 🎓 School Management System

A full-stack School Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js). The platform helps schools manage students, staff, attendance, departments, fees, and administrative operations efficiently through a centralized dashboard.

## 🚀 Features

### 🔐 Authentication & Authorization

* JWT Authentication
* Role-Based Access Control (RBAC)
* Super Admin Login
* School Admin Login
* Protected Routes

### 🏫 School Management

* Add School
* Update School
* Delete School
* View All Schools
* School Admin Creation

### 👨‍🏫 Staff Management

* Add Staff
* Update Staff
* Delete Staff
* Staff Type Management
* Department Assignment

### 🎓 Student Management

* Add Students
* Update Students
* Delete Students
* Student Records Management

### 📋 Attendance Management

* Mark Attendance
* Present/Absent Tracking
* Attendance Reports

### 💳 Fees Management

* Fee Collection Tracking
* Payment Records
* Due Monitoring

### 📊 Dashboard

* Total Schools
* Total Staff
* Total Students
* Total Departments
* Total Staff Types
* Attendance Statistics
* Recent Activities

### 🤖 AI Assistant

* AI-powered assistance module
* School-related query support

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/abhi-148/Schoolmanagmenetsystem.git
cd Schoolmanagmenetsystem
```

### Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
EMAIL_USER=your_email
EMAIL_PASS=your_password
```

Start Backend:

```bash
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
